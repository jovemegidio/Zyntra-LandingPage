Gerenciar Treinamentos

Adicione os IDs dos vídeos do YouTube para cada aula. Clique em "Publicar" para salvar as alterações.

Como adicionar vídeos: Faça upload no YouTube (público ou não-listado) → Copie o ID da URL: youtube.com/watch?v=VIDEO_ID → Cole no campo "ID do Vídeo" da aula correspondente → Clique em "Publicar alterações". As alterações ficam salvas no navegador e são aplicadas imediatamente na página de treinamentos.
